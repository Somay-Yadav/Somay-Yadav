## 🎥 TubeFetch

A fast and user-friendly command-line YouTube downloader built with Python. TubeFetch allows users to download videos and audio, view video information, choose quality options, and monitor download progress through an interactive terminal interface.

### Features

* 🎬 Download YouTube videos
* 🎵 Download audio only
* 📺 Display video information
* ⚡ Multiple quality options
* 📊 Real-time download progress
* 🎨 Interactive CLI with ASCII interface
* 🔄 Continuous download session
* 🖥️ Cross-platform support
